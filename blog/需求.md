### 方案 B：Obsidian 单独仓库，用脚本同步

如果你不想动现有的 Obsidian 结构，写一个同步脚本：

```python
# sync_obsidian_to_blog.py
import os
import shutil
import frontmatter  # pip install python-frontmatter
from datetime import datetime

OBSIDIAN_VAULT = "/path/to/your/obsidian/vault"
BLOG_DIR = "/path/to/my-tech-blog/blog"

def sync_articles():
    for root, dirs, files in os.walk(OBSIDIAN_VAULT):
        for file in files:
            if not file.endswith('.md'):
                continue
            
            src_path = os.path.join(root, file)
            
            # 只同步标记为 publish: true 的文章
            with open(src_path, 'r', encoding='utf-8') as f:
                post = frontmatter.load(f)
            
            if not post.metadata.get('publish', False):
                continue
            
            # 构造 Docusaurus 兼容的文件名
            date = post.metadata.get('date', datetime.now())
            title = post.metadata.get('title', file.replace('.md', ''))
            slug = title.lower().replace(' ', '-')
            dest_name = f"{date.strftime('%Y-%m-%d')}-{slug}.md"
            
            # 添加 Docusaurus 必需的 frontmatter
            post.metadata['slug'] = slug
            post.metadata['tags'] = post.metadata.get('tags', [])
            
            # 写入博客目录
            dest_path = os.path.join(BLOG_DIR, dest_name)
            with open(dest_path, 'w', encoding='utf-8') as f:
                f.write(frontmatter.dumps(post))
            
            print(f"✅ Synced: {file} → {dest_name}")

if __name__ == "__main__":
    sync_articles()
```

## 你关心的几个问题

### 国内访问速度

- Vercel 国内节点（北京/上海/深圳）速度不错
- 如果绑定自定义域名走 Cloudflare 代理，延迟约 30-50ms
- 实测个人博客完全够用

### 图片处理

Obsidian 里的图片路径和 Docusaurus 不同，建议：

1. 图片统一放到 `static/img/` 目录
2. 文章中引用：`![渲染管线图](/img/opengl-pipeline.png)`
3. 或者用图床（阿里云 OSS / 腾讯云 COS）+ CDN

### 数学公式

Docusaurus + Katex 效果：

```latex
$$
L_o(p,\omega_o) = \int_{\Omega} 
L_i(p,\omega_i) f_r(p,\omega_i,\omega_o) 
(\omega_i \cdot n) d\omega_i
$$
```

渲染结果：$L_o(p,\omega_o) = \int_{\Omega} L_i(p,\omega_i) f_r(p,\omega_i,\omega_o) (\omega_i \cdot n) d\omega_i$

## 一键初始化脚本

我帮你写了一个完整的初始化脚本，保存为 `setup.sh`：

```bash
#!/bin/bash

# 博客名称
BLOG_NAME="my-tech-blog"

# 1. 创建项目
npx create-docusaurus@latest $BLOG_NAME classic --typescript
cd $BLOG_NAME

# 2. 安装插件
npm install remark-math rehype-katex
npm install prism-react-renderer

# 3. 创建目录结构
mkdir -p static/img
mkdir -p src/components
mkdir -p blog

# 4. 写入配置
cat > docusaurus.config.ts << 'EOF'
// 复制上面提供的完整配置
EOF

# 5. 创建示例文章
cat > blog/2024-01-01-welcome.md << 'EOF'
---
title: 欢迎来到我的图形学技术博客
date: 2024-01-01
tags: [hello, graphics]
---

## 关于我

C++ 图形图像开发，主要技术栈：
- OpenGL / Vulkan
- VTK
- Qt
- 计算几何

## 文章分类

- **OpenGL**：渲染管线、着色器、高级渲染技术
- **VTK**：体绘制、网格处理、可视化算法
- **Qt**：OpenGL Widget 集成、自定义控件
- **计算几何**：几何算法、三角剖分、曲面重建

$$
f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi) e^{2\pi i \xi x} d\xi
$$
EOF

# 6. 初始化 git
git init
git add .
git commit -m "init: 图形学技术博客"

echo "✅ 博客创建完成！"
echo "接下来："
echo "1. cd $BLOG_NAME"
echo "2. npm run start  # 本地预览"
echo "3. 推送到 GitHub 并关联 Vercel"
```
